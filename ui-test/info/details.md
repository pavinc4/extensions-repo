# UI Zone Tester

Dev tool for testing all injectable UI zones in the platform detail page.

## Zones tested

- `detail_actions` — buttons injected into the action bar (Open Config, View Logs)
- `detail_tabs` — extra tabs added to the tab bar (App, Logs)
- `side_panel` — info panel on the left of tab content
- `status_bar` — status strip at the bottom of the detail page

Delete this tool once the layout is locked and zones are confirmed working.
