# Any Downloader

Download audio and video from YouTube, Instagram, Twitter, Reddit, and 1000+ sites.

When enabled, a popup window opens. Paste any URL to get started.

## Audio

- **320 kbps MP3** — best quality (shown when source is 200kbps+)
- **128 kbps MP3** — smaller file size
- **WAV** — lossless uncompressed
- **Original** — source format, no conversion

## Video

- All available resolutions shown dynamically
- Merged as MP4 automatically
- Audio quality scales with resolution

## Requirements

None. yt-dlp is installed automatically on first run.
